FiveM Roleplay UI

Комплексный пользовательский интерфейс для ролевых серверов FiveM с системой инвентаря, HUD, диалогов и кастомизацией персонажа. Проект разработан с использованием Lua и JavaScript, с акцентом на производительность и удобство использования.

Основные особенности:

  • Полностью настраиваемый интерфейс
  • Система инвентаря с drag-and-drop
  • Динамический HUD с показателями здоровья, брони и голода
  • Система диалогов с ветвлениями
  • Кастомизация персонажа в реальном времени
  • Оптимизированная производительность
