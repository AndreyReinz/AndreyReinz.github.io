Обо мне

Приветствую! Меня зовут Andrey Reinz, и я профессиональный фронтенд-разработчик с более чем 5-летним опытом создания современных веб-приложений. Моя страсть к программированию сочетается с увлечением игровой индустрией, что привело меня к специализации на разработке модов для GTA V на платформах FiveM и RAGE MP.

Мой путь в IT начался с изучения веб-технологий, где я освоил весь стек современных фронтенд-инструментов. Со временем я обнаружил, что могу сочетать свои навыки веб-разработки с любовью к играм, создавая сложные системы для ролевых серверов GTA V.

Особое внимание я уделяю созданию интуитивно понятных пользовательских интерфейсов, которые значительно улучшают игровой опыт. Мои системы отличаются стабильностью, производительностью и вниманием к деталям.

В последнее время я сосредоточился на разработке комплексных решений для ролевых серверов, включая системы регистрации оружия с админ-панелью, duty-системы для фракций, кастомные экономические системы и многое другое. Каждый мой проект - это тщательно продуманное решение, учитывающее потребности как игроков, так и администраторов серверов.

Я постоянно совершенствую свои навыки, изучаю новые технологии и подходы к разработке. Моя цель - создавать продукты, которые не только функциональны, но и доставляют удовольствие от использования.

Мои проекты

Система регистрации оружия

Комплексное решение для регистрации и отслеживания оружия на ролевых серверах GTA V с расширенной админ-панелью и логированием действий.

Основные функции:

  • Регистрация оружия с привязкой к персонажу
  • Полноценная админ-панель с разграничением прав
  • Система проверок и аудита
  • Интеграция с экономической системой сервера
  • Логирование всех действий с оружием
  • Кастомный UI для игроков и администраторов
FiveM Lua JavaScript React MySQL

Duty системы для фракций

Набор кастомных duty-систем для различных фракций (полиция, EMS, государственные организации) с уникальными функциями для каждой.

Основные функции:

  • Индивидуальные системы для каждой фракции
  • Автоматическое снаряжение при входе в duty
  • Система рангов и привилегий
  • Интеграция с другими системами сервера
  • Кастомные интерфейсы для разных фракций
  • Система отчетности и статистики
RAGE MP C# JavaScript Vue.js MongoDB

RP Server UI Framework

Фреймворк для создания пользовательских интерфейсов на ролевых серверах с поддержкой тем оформления и модульной архитектурой.

Основные функции:

  • Готовые UI-компоненты для RP-серверов
  • Поддержка тем оформления (light/dark/custom)
  • Адаптация под разные разрешения экрана
  • Интеграция с основными RP-системами
  • Модульная архитектура для легкого расширения
  • Оптимизация производительности
